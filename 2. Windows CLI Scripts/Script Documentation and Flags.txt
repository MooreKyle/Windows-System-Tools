======================= 0 - Cancel Pending Repairs and Maintenance =======================
Purpose:
Cancels maintenance tasks, scheduled shutdowns, and diagnostic flags set to run after reboot.
Built for safety and clarity ‚Äî no registry interaction. Cancels any scheduled PC shutdown
or restart at the start of the script for ease of access (if safety checks prevent 2nd one).

Commands Used:
- shutdown /a                           ‚Äì Cancels any scheduled shutdown or restart
- tasklist | findstr TiWorker.exe...    ‚Äì Detects if Windows Update is actively running
- netstat -b | findstr bits             ‚Äì Checks for BITS (background update) activity
- chkntfs /x C:                         ‚Äì Excludes drive C: from automatic CHKDSK on boot
- shutdown /a (x2)                      ‚Äì Cancels any scheduled shutdown or restart

Requires Admin Privileges: ‚úÖ
Requires Network Connection: ‚ùå



================================ 1.4 - System Performance ================================

============================= 1.8 - System Health and Repair =============================
Purpose:
Scans and repairs Windows system files using DISM and SFC.
Also checks drive C: for pending file system errors and repair flags.
Logs results and log results/locations for most commands that are run,
and notifies the user.

Commands Used:
- sfc /scannow                        ‚Äì Scans and repairs corrupted or missing system files
- dism /online /cleanup-image /checkhealth      ‚Äì Checks Windows image for known corruption
- dism /online /cleanup-image /restorehealth  ‚Äì Repairs detected image issues (requires Internet)
- chkdsk /f                    ‚Äì Scans and fixes file system errors on next reboot (drive C:)
- chkdsk /r                  ‚Äì Scans for bad sectors (includes /f, runs on reboot) (drive C:)
- fsutil dirty query C:        ‚Äì Reports if drive C: is flagged for CHKDSK at next boot

Requires Admin Privileges: ‚úÖ
Requires Network Connection: üì∂ (for /restorehealth only)
